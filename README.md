# Nexus Calculator Suite

A professional multi-tab calculator application built with Python and `ttkbootstrap`.

## Features

- **DCA Breakeven**: Calculate Dollar-Cost Average breakeven price and take profit targets.
- **Data Plan Comparator**: Compare different data plans to find the best value (NGN/GB, NGN/day).
- **Compound Interest**: Project growth with flexible compounding frequencies and export steps to CSV.
- **Percent Change**: Calculate percentage increase/decrease and maintain a mini history.

## Installation

Ensure you have Python installed. You can install the dependencies using `pip` or `uv`.

### Using pip
```bash
pip install ttkbootstrap
```

### Using uv
```bash
uv pip install ttkbootstrap
```

## How to Run

Navigate to the project directory and run:

```bash
python calculator_suite.py
```
